#DeusBOT

DeusBOT — это интеллектуальный погодный бот, который предоставляет точные прогнозы погоды и персонализированные уведомления для пользователей.

О проекте

▸ Поддержка различных единиц измерения

▸ Интерактивные уведомления с рекомендациями

▸ Поддержка нескольких языков (в разработке)

▸ Геймификация и достижения (в будущем обновлении)

Установка и запуск

Клонируйте репозиторий:
```bash
git clone https://github.com/username/DeusBOT.git
```
Установите зависимости:
```bash
pip install -r requirements.txt
```
Запустите бота:
```bash
python bot.py
```
